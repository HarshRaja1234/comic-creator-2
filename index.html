Comic Creator

Using the Comic Creator, anyone can make and share cool 10-panel comic strips in a simple interface. It works by using a special tool that turns text into pictures. So, when you write a story, this tool transforms it into awesome comic panels.

If you leave a panel empty (without writing anything), it will show a message saying 'Panel text is empty' for that specific panel.
